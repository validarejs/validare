# Plugins — Reference

All 9 built-in plugins.

## Core Plugins

| Plugin | Key | Description |
|---|---|---|
| [`Trigger`](./Trigger.md) | `trigger` | Validate on DOM events |
| [`Message`](./Message.md) | `message` | Display error messages |
| [`Icon`](./Icon.md) | `icon` | Show validation state icons |
| [`SubmitButton`](./SubmitButton.md) | `submitButton` | Disable submit during validation |
| [`Excluded`](./Excluded.md) | `excluded` | Exclude disabled/hidden fields |
| [`Sequence`](./Sequence.md) | `sequence` | Stop at first error per field |

## CSS Framework Plugins

| Plugin | Key | Framework |
|---|---|---|
| [`Bootstrap5`](./Bootstrap5.md) | `ui` | Bootstrap 5 |
| [`Bulma`](./Bulma.md) | `ui` | Bulma |
| [`Tailwind`](./Tailwind.md) | `ui` | Tailwind CSS |

> **Note:** All three CSS framework plugins share the key `ui`. Register only one per form instance.
